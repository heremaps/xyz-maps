Interface LinearGradient

Represents a linear gradient with color stops.

example
const gradient: LinearGradient = {
    type: 'LinearGradient',
    stops: {
        1.0: 'white',
        0.9: '#FCFBAE',
        0.8: '#FAD932',
        0.7: '#F26C19',
        0.5: '#C41D6F',
        0.3: '#70009C',
        0.0: '#1E0073'
    }
};

Properties

stops

stops: {}

Defines the color stops for the gradient. Keys represent stop positions (ranging from 0.0 to 1.0), where 0.0 is the start and 1.0 is the end of the gradient.

Type declaration

  • [stop: number]: string

type

type: "LinearGradient"

Specifies the gradient type. Must always be "LinearGradient".
